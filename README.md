# cs367_JimLoey
ระบบจองคิวร้านอาหารและคาเฟ่ ช่วยให้ลูกค้าสามารถจองคิวและตรวจสอบสถานะคิวได้ล่วงหน้า ลดเวลารอคิวหน้าร้าน และช่วยให้ร้านค้าบริหารจัดการคิวได้อย่างเป็นระบบ สะดวก รวดเร็ว และเพิ่มประสบการณ์ที่ดีให้ทั้งลูกค้าและผู้ให้บริการ

### ผู้รับผิดชอบ นางสาว อนุตตมา แสงทอง 6609650731

> POST /auth/register ใช้สำหรับสมัครสมาชิกใหม่ (Customer) เข้าสู่ระบบ

> POST /auth/login ใช้สำหรับเข้าสู่ระบบ

> GET /restaurants/{res_id}/reports/booking/summary ใช้สำหรับดูสถิติภาพรวม เช่น จำนวนการจองทั้งหมด ยกเลิก สำเร็จรายได้รวม ตามช่วงวันที่กำหนด

### ผู้รับผิดชอบ นางสาว สิฏฐิยา สง่าเมือง 6609650681

> POST /booking ใช้สำหรับสร้างการจองโต๊ะใหม่ โดยระบุร้าน วันที่ เวลา และจำนวนคน

> GET /booking/{user_id} ใช้สำหรับดูรายละเอียดการจองแต่ละรายการของผู้ใช้

### ผู้รับผิดชอบ นาย ภคิน มณีรัตน์ 6609650558

> UPDATE /{res_id}/booking/{book_id}/status ใช้สำหรับอัปเดตสถานะการจอง เช่น อนุมัติ ปฏิเสธ

> GET /{res_id}/booking ใช้ตรวจสอบรายการจอง

### ผู้รับผิดชอบ นางสาว ดวงกมล แก้วสาลาภูมิ 6609650384

> GET /search ใช้สำหรับค้นหาร้านอาหาร

> GET /search/{res_id} ใช้สำหรับดูรายละเอียดร้าน

### ผู้รับผิดชอบ นางสาว ชัญญานุช อิ่มเกิด 6609650277

> POST /restaurants ใช้สำหรับให้ผู้ประกอบการเพิ่มร้านอาหารใหม่เข้าสู่ระบบ

> GET /bookings/queue ใช้สำหรับดูคิวทั้งหมด ณ ขณะนั้น

### ผู้รับผิดชอบ นางสาว พีรดา งามวงศ์วิโรจน์ 6609650533

> PATCH /tables/{table_id}/status ใช้สำหรับการเปลี่ยนสถานะโต๊ะ

> GET /tables/count ใช้ดูจำนวนโต้ะทั้งหมด
